# 如何正确打开并显示网站

## 方法一：直接打开（推荐）

项目已改为**相对路径**，用浏览器直接打开 HTML 即可正常显示样式和图片。

1. 在 Finder 中进入项目里的 **en**（英文）、**cn**（中文）或 **ru**（俄文）文件夹。
2. **双击** `index.html` 用默认浏览器打开首页；或右键 → 用 Chrome / Safari 等打开。
3. 其他页面（如 `contact.html`、`valves.html`）同样直接双击打开即可。

注意：一定要从 **en / cn / ru 文件夹里** 打开对应的 HTML，不要从别处打开，否则相对路径会错。

---

## 方法二：用本地服务器打开

在终端中进入项目根目录后执行：

```bash
# Python 3
python3 -m http.server 8080

# 或使用 npx（需已安装 Node.js）
npx serve -l 8080
```

然后在浏览器访问：

- 英文首页: http://localhost:8080/en/index.html  
- 中文首页: http://localhost:8080/cn/index.html  
- 联系页: http://localhost:8080/en/contact.html  

按 `Ctrl+C` 可停止服务器。
