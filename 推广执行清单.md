# www.fuish.cn 推广执行清单

面向石油天然气行业的 B2B 网站（阀门、法兰、管道解决方案），多语言（中/英/俄/西），配合 NEFTEGAZ 2026 展会（莫斯科，展台 10C018）。

---

## 一、搜索引擎（必做，优先）

### 1. Google

- [ ] **Google Search Console**  
  打开 [search.google.com/search-console](https://search.google.com/search-console) → 添加资源 `https://www.fuish.cn` → 用 HTML 文件或域名 DNS 验证 → 提交 sitemap：`https://www.fuish.cn/sitemap.xml`
- [ ] 提交后定期查看「覆盖率」「效果」：看哪些页面被收录、哪些关键词有展示/点击，据此优化标题与描述。

### 2. Yandex（俄罗斯/独联体必做）

- [ ] **Yandex.Webmaster**  
  [webmaster.yandex.ru](https://webmaster.yandex.ru/) → 添加站点 `https://www.fuish.cn` → 验证所有权 → 提交 sitemap：`https://www.fuish.cn/sitemap.xml`
- [ ] **Yandex.Metrika**  
  [metrika.yandex.ru](https://metrika.yandex.ru/) → 为 www.fuish.cn 创建计数器 → 把统计代码插入全站（至少 `/ru/` 所有页），用于分析俄语访客来源与关键词。
- [ ] 在 Webmaster 中关注索引、抓取错误、部分搜索查询数据。

### 3. Bing（可选，部分国际流量）

- [ ] **Bing Webmaster Tools**  
  [bing.com/webmasters](https://www.bing.com/webmasters) → 添加站点并验证 → 提交 `https://www.fuish.cn/sitemap.xml`。

---

## 二、B2B 与行业平台

- [ ] **国内 B2B**：在阿里巴巴国际站、中国制造网、慧聪等创建/更新公司页，简介中写清产品（阀门、法兰、石油天然气管道），并放官网链接 https://www.fuish.cn 及 NEFTEGAZ 2026 展台信息。
- [ ] **国际/俄罗斯 B2B**：在 TradeKey、ExportHub、俄罗斯/独联体行业目录或展会合作平台（如 NEFTEGAZ 参展商列表）填写公司信息，链接到 www.fuish.cn 对应语言首页（俄语用 `/ru/index.html`）。
- [ ] **LinkedIn**：公司主页与关键人员资料中统一写官网、产品线、NEFTEGAZ 2026 展台 10C018，重要动态可带官网链接。

---

## 三、展会与线下联动（NEFTEGAZ 2026）

- [ ] **展会官网/参展商页**：若 NEFTEGAZ 提供参展商链接，填写 https://www.fuish.cn（或俄语首页），并写清展台 10C018。
- [ ] **物料统一**：名片、易拉宝、宣传册上的网址统一为 www.fuish.cn，可加「俄语入口：www.fuish.cn/ru/」。
- [ ] **邮件签名**：销售/市场人员邮件签名中放官网链接 + 「NEFTEGAZ 2026 · 莫斯科 · 展台 10C018」。

---

## 四、内容与社交（持续）

- [ ] **简短新闻稿/动态**：发布「FUI 参展 NEFTEGAZ 2026」等消息，文中带 https://www.fuish.cn 及产品页链接，可投行业媒体或自家公众号/站内新闻（若后续有新闻页）。
- [ ] **社交媒体**：如有 LinkedIn / Facebook / 微信公号，发展会与产品内容时附官网链接；俄语市场可考虑 VK 等。
- [ ] **落地页**：推广时尽量链到具体页面（如俄语阀门页、法兰页、联系页），便于统计与转化。

---

## 五、技术与小优化（配合推广）

- [ ] **robots.txt**：确认已有且包含 `Sitemap: https://www.fuish.cn/sitemap.xml`（当前项目应已具备）。
- [ ] **俄文 title/description**：在主要俄文页的 title、meta description 中自然融入「нефтегаз」「клапаны」「фланцы」「НЕФТЕГАЗ 2026」等词（详见项目内《SEO-俄罗斯市场建议》）。
- [ ] **打开速度**：压缩图片、减少大文件，有条件的可考虑对俄罗斯用户使用离俄较近的 CDN。

---

## 六、执行顺序建议

| 阶段   | 建议先做 |
|--------|----------|
| 第 1 步 | Google Search Console + 提交 sitemap |
| 第 2 步 | Yandex Webmaster 验证 + 提交 sitemap + 安装 Yandex.Metrika |
| 第 3 步 | B2B 平台更新公司信息与官网链接，展会物料与邮件签名统一网址 |
| 第 4 步 | 新闻稿/社交发稿 + 持续根据 Metrika/Search Console 数据优化关键词与落地页 |
